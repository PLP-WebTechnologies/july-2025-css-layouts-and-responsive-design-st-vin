Introduction

Python has several built-in datatypes that help you store and work with different kinds of data. Here are the most important ones for beginners:

int

Stores whole numbers, like 5 or -3.

float

Stores decimal numbers, like 3.14 or -0.5.

str

Stores text, like "Hello" or 'Python'.

bool

Stores True or False values for logic.

list

Stores a collection of items, like [1, 2, 3] or ["a", "b"].

tuple

Stores a fixed collection of items, like (1, 2).

dict

Stores key-value pairs, like {"name": "Alice", "age": 20}.

set

Stores unique items, like {1, 2, 3}.

Tips for Beginners

  • Use type() to check a variable's datatype.
  • Strings are always in quotes.
  • Lists and tuples can store any datatype.
  • Dictionaries use curly braces and key-value pairs.
  • Practice by creating variables of each type!
